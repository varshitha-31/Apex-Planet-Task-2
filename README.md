# Apex Planet - Task 2
## Exploratory Data Analysis (EDA) & Business Intelligence

### 📌 Project Overview
This project is part of the Apex Planet Data Analytics Internship. The objective is to perform Exploratory Data Analysis (EDA), write SQL queries to answer business questions, and create a dashboard using a retail sales dataset.

### 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- SQL
- Google Colab

### 📊 Tasks Performed
- Imported and explored the dataset
- Performed descriptive statistics
- Checked missing values
- Created visualizations:
  - Histogram (Age Distribution)
  - Bar Chart (Product Category)
  - Scatter Plot (Age vs Total Sales)
  - Pair Plot
  - Correlation Heatmap
  - Monthly Sales Trend
- Wrote SQL queries to answer business questions
- Created an interactive dashboard

### 📈 Business Questions Answered
1. Total Revenue
2. Top 5 Products by Revenue
3. Highest Sales City
4. Best Performing Category
5. Gender-wise Revenue Analysis
6. Highest Spending Customer
7. Average Order Value
